MultipleProjects(多项目) [汉化反馈]

&productName; 可以在一个窗口中操作多个项目。打开您的项目，而另一个已打开。然后，根据 Settings/Preferences 的 System Settings 页面的 Project Opening 部分中选择的选项，会发生以下情况：

  • 如果选择了 Open project in a new window 选项，则新项目将在新窗口中静默打开。Attach Project 选项出现在 File 菜单中。
  • 如果选择了 Open projects in the same window 选项，则新项目将在同一窗口中静默打开，替换当前打开的项目。Attach Project 命令出现在 File 菜单中。
  • 如果选择了 Confirm window to open project in 选项，&productName; 会提示您选择是否要在新窗口中打开项目，重新使用现有窗口，或者将新项目附加到当前打开的项目。已经打开的项目是主项目。
