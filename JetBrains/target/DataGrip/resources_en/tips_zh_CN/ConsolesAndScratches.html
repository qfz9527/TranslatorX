ConsolesAndScratches(控制台和临时文件) [汉化反馈]

Database consoles 是 SQL 文件，您可以在其中编写和执行 SQL 语句。与临时文件不同，控制台附加到数据源。创建数据源时，会自动创建数据库控制台。要打开控制台，请右键单击数据源，然后选择 Open Console(或按 &shortcut:EditSource;)。

Scratch files 类似于控制台，但它们未附加到数据源。请将临时文件作为临时说明或草稿，用于在项目上下文之外的某些代码构思中。要创建临时文件，请导航到 File | New | Scratch File(或按 &shortcut:NewScratchFile;)。

Scratch buffers 是简单的文本文件，没有任何编码辅助功能。Scratch 缓冲区可用于简单的任务列表和自己的注释。创建新暂存缓冲区的操作没有专用菜单项，但您可以使用 Find Action 弹出窗口(&shortcut:GotoAction;)并运行 New Scratch Buffer 动作。
